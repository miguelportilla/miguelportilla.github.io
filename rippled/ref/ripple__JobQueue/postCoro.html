JobQueue::postCoro

Creates a coroutine and adds a job to the queue which will run it.

Synopsis

    class F>

    JobType t,

    F&& f);
Description
Parameters

Name

Description

The type of job.

name

Name of the job.

f

Has a signature of void(std::shared_ptr<Coro>). Called when the job executes.
